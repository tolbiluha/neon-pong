# Неон Понг

Мобільна браузерна гра для двох гравців на одному телефоні.

## Грати онлайн

Після ввімкнення GitHub Pages гра доступна за адресою:

https://tolbiluha.github.io/neon-pong/

## Керування

- Гравець 1 рухає пальцем по лівій половині екрана.
- Гравець 2 — по правій.
- Матч триває до 7 очок.
- На комп'ютері також працюють клавіші `W` / `S` і `↑` / `↓`.
